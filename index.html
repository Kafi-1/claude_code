🤖 Kafi Claude Code Guide

Termux দিয়ে Claude Code ও Chat App সেটআপ করুন

✅ Full Setup Guide
⚙️ Part 1: Claude Code Termux Setup
Termux এ Claude Code ইন্সটল ও চালু করুন
1️⃣ Update & Upgrade 
            pkg update && pkg upgrade
2️⃣ Install Dependencies 
            pkg install nodejs git nano
3️⃣ Check Versions 
            node -v
          
            npm -v
4️⃣ Install Claude Code 
            npm install -g @anthropic-ai/claude-code@2.1.112
5️⃣ Run Installer 
            node $PREFIX/lib/node_modules/@anthropic-ai/claude-code/install.cjs
6️⃣ Environment Variables 
            export ANTHROPIC_BASE_URL=https://agentrouter.org/
          
            export ANTHROPIC_AUTH_TOKEN=sk-Z6Bxxxxxxxxxxxxxxxx (REPLACE WITH YOUR ACTUAL API KEY)
          
            export ANTHROPIC_MODEL=claude-opus-4-6
          
            export CLAUDE_CODE_USE_AUTH_TOKEN=true
7️⃣ Run Claude 
            claude
💬 Part 2: Claude Chat App Setup
index.html + server.js দিয়ে Browser-based Chat চালান
1️⃣ প্রজেক্ট ফোল্ডার তৈরি করুন 
            mkdir ~/claude-chat && cd ~/claude-chat
2️⃣ ফাইল Download করুন নিচে থেকে 👇

নিচের দুটো ফাইল Download করে claude-chat ফোল্ডারে রাখুন:

🟨
server.js
Node.js backend server — Claude Code CLI চালায়

এই ফাইলটি ~/claude-chat/ ফোল্ডারে রাখুন

⬇️ Download

📁 GitHub থেকে: uploads/server.js

🟦
index.html
Chat UI — Browser এ চলে, Claude এর সাথে কথা বলুন

এই ফাইলটি ~/claude-chat/ ফোল্ডারে রাখুন

⬇️ Download

📁 GitHub থেকে: uploads/index.html

3️⃣ Environment Variables সেট করুন (প্রতি session এ) 
            export ANTHROPIC_BASE_URL=https://agentrouter.org/
          
            export ANTHROPIC_AUTH_TOKEN=sk-xxxxxxxxxxxxxxxx
          
            export ANTHROPIC_MODEL=claude-opus-4-6
          
            export CLAUDE_CODE_USE_AUTH_TOKEN=true
4️⃣ Server চালু করুন 
            cd ~/claude-chat && node server.js

✅ সফল হলে দেখবেন: Claude Chat running: http://localhost:3000

5️⃣ Browser এ খুলুন 
            http://localhost:3000

📱 Termux এ server চলতে থাকবে, ফোনের যেকোনো browser এ উপরের link খুলুন।

⚠️ গুরুত্বপূর্ণ টিপস
  • Server বন্ধ করতে Termux এ Ctrl+C চাপুন
  • নতুন session এ environment variables আবার export করতে হবে
  • সব সময় claude-chat ফোল্ডারে থেকে server চালু করুন
  • Port 3000 ব্যবহার করতে না পারলে: PORT=8080 node server.js
  • Termux এ server চলুক, আলাদা browser tab এ chat করুন
